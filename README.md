# Feedbacknetz
 
Ein interaktives Spinnendiagramm-Tool zur visuellen Darstellung und zum Vergleich persönlicher und fachlicher Einschätzungen.
 
---

## Verwendung
 
Datei `feedbacknetz.html` herunterladen und im Browser öffnen. Keine Installation, kein Server, keine Abhängigkeiten.
 
---
 
## Funktionsumfang
 
**Bewertung und Eingabe**
 
Kategorien werden direkt per Klick im Diagramm bewertet. Über ein Namensfeld lässt sich jede Einschätzung beschriften. Das fertige Diagramm kann als PNG exportiert werden.
 
**Kategorien und Skalierung**
 
Achsen können umbenannt, hinzugefügt und entfernt werden. Die Skalierung ist zwischen 3 und 15 Stufen frei einstellbar.
 
**Darstellungsmodi**
 
| Modus | Beschreibung |
|---|---|
| Standard | Klassisches Spinnendiagramm mit Skalengitter |
| Diskret | Nur Strahlen, kein Gitter |
| Invertiert | Skala umgekehrt — hohe Werte wahlweise innen oder außen |
| Entwicklung | Zwei Zustände gegenüberstellen, z. B. *vorher* und *nachher* |
 
---
 
## Umsetzung
 
Das Tool ist als einzelne `.html`-Datei konzipiert — HTML, CSS und JavaScript ohne Build-Schritt, ohne externe Abhängigkeiten. Entstanden durch einen iterativen Prozess mit [Claude](https://claude.ai), [GitHub Copilot](https://github.com/features/copilot) und manuellem Feinschliff.
